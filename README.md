# Documate

## 프로젝트 목표

- 한국어 매뉴얼 문서를 기반으로, 사용자가 질문하면 문서 내용만 참고해 자연스러운 대화체로 답변하는 챗봇 구현
- PDF는 텍스트와 이미지가 혼합된 구조를 지원하며, 할루시네이션(근거 없는 답변)을 완전히 차단하고 문서에 없는 내용은 사용자가 지정한 답변을 할 수 있도록 응답
- 목표 응답 속도는 5초 이내
- React - FastAPI - Local LLM 구조의 웹 기반 서비스 제공

### 추가 프로젝트 목표

- 매뉴얼에서 끝나는게 아닌, 어떤 특정한 PDF 문서를 올리더라도 그것에 관한 질문 답변을 철저한 레퍼런스와 함께 검증된 답변을 제공

---

## 핵심 컨셉

> 딱딱한 기술 매뉴얼을 대화형으로 풀어주는 AI 도우미, 문서 기반 및 신뢰도 중심
